Пет-проект AllParkings
github.com/pavelkulak/AllParkings
Fullstack разработчки
AllParkings – веб-приложение для поиска и бронирования парковок
Описание: Разработал интерактивное веб-приложение, позволяющее пользователям находить,
бронировать и управлять парковочными местами. Реализована карта с поиском парковок,
система бронирования, отзывы, чат и панель администратора для управления объектами.
Стек технологий:
Frontend: React 18, TypeScript, Redux Toolkit, React Router, Material UI, 2GIS Map API, Socket.io
Client, Axios, Vite.
Backend: Node.js, Express, Sequelize (PostgreSQL), Socket.io, JWT, Multer, Bcrypt.
Ключевые функциональности:
Интерактивная карта парковок с API 2GIS.
Поиск, фильтрация и бронирование парковочных мест.
Авторизация и личный кабинет пользователя.
Чат для взаимодействия пользователей и владельцев парковок.
Панель администратора для управления парковками.
Система отзывов, рейтингов и избранных парковок.
Результат: Улучшение пользовательского опыта при поиске парковки, повышение
заполняемости парковочных мест за счет удобного бронирования.
